# Bonuspay Checkout - 前端开发技术规范

## 1. 技术栈 (Tech Stack)
*   **Core:** Vite + React + TypeScript
*   **UI Framework:** Ant Design (Antd) - 用于基础组件结构
*   **Styling:** TailwindCSS - **必须覆盖 Antd 的默认样式**，去除企业级边框，实现圆角、柔和阴影。
*   **Web3:** Wagmi + Viem + @tanstack/react-query

## 2. 视觉风格 (Visual Guidelines)
*   **关键词:** 简洁、商业化、智能感。
*   **动效 (Tailwind Config):**
    *   `animate-float`: 用于图标悬停。
    *   `animate-breathe`: 用于选中状态的光晕呼吸。
    *   `animate-radar`: 用于兜底页面的监听信号。

## 3. 核心逻辑 (Key Logic)
*   **EIP-6963:** 使用 wagmi `useConnectors` 自动探测插件。
*   **Mock API:** 模拟 Binance Pay 的请求（axios 延时返回成功）。
*   **Auto-Fallback:** 全局监听连接状态，错误发生时自动改变 UI 状态机，禁止使用 `window.alert`。
